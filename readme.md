### Sistema de Gestión de Inventario con Login

## Esta aplicación es un sistema básico de gestión de inventario desarrollado en PHP y MySQL, diseñado para administrar productos de manera sencilla y eficiente. Incluye un sistema de autenticación de usuarios (login) para garantizar el acceso seguro a la información.

 ### Características principales
 ## Sistema de autenticación
 ## Inicio de sesión para usuarios registrados
 ## Protección de rutas para evitar accesos no autorizados
 ## Gestión de productos
 ## Crear nuevos productos
 ## Listar productos existentes
 ## Editar información de productos
 ## Eliminar productos del inventario
 ## Base de datos MySQL
### Almacenamiento estructurado de usuarios y productos
## Relación entre tablas para mejor organización
 ## Interfaz sencilla
## Navegación clara y fácil de usar
## Formularios básicos para la gestión de datos
## Tecnologías utilizadas
## PHP (backend)
## MySQL (base de datos)
## HTML5 / CSS3 (estructura y estilos)
## JavaScript (opcional para validaciones o mejoras UI)
