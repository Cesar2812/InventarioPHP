Sistema de Inventario Básico - PHP + MySQL + PDO
Descripción

Sistema web básico de inventario desarrollado en PHP utilizando MySQL como gestor de base de datos y PDO para la conexión segura a la base de datos.
Permite administrar productos, categorías y controlar existencias de manera sencilla.

El sistema incluye un módulo de autenticación básico con inicio de sesión y protección de rutas para evitar el acceso no autorizado a las páginas internas del sistema.

Este proyecto está orientado a prácticas académicas, aprendizaje de PHP y como base para futuros sistemas más completos.

Características
Inicio de sesión de usuarios
Protección de rutas mediante sesiones
Registro de productos
Edición y eliminación de productos
Gestión de categorías
Control de stock
Listado de productos
Búsqueda de productos
Conexión segura mediante PDO
Arquitectura básica organizada
Uso de consultas preparadas para mayor seguridad
